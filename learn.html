Four rules.
A visual grammar for arithmetic.

01 —

The Number

One circle = one number.

Each integer is represented as a circle made of concentric rings. The number of rings equals the number of digits.

Each ring encodes one digit — the outermost ring holds the most significant digit. The arc that is filled corresponds to the digit's value: a full circle is 9, an empty ring is 0.

Positive — arcs are filled solid.
Negative — arcs are hollow (stroke only).

02 —

The Operation

Addition as a belt.

When two numbers are added, a belt is drawn between their circles. The belt is formed by two external tangent lines and the arcs that connect them — wrapping both circles in one continuous shape.

The result is a third circle, placed at the end of the chain. Addition becomes a spatial relationship, not a symbol.

03 —

The Plane

Multiplication as intersection.

Multiplication uses the Japanese line method — each digit becomes a group of parallel lines. Lines for the first number run diagonally in one direction; lines for the second number run across them.

The intersections are counted in groups. Each group corresponds to a digit in the result — the geometry of the product.

04 —

Colour

Each number: its own palette.

Colour in this system is not decoration — it is identification. Each digit ring is assigned a colour from a curated palette of 28 hand-picked HSB swatches.

The palette is divided into light and dark values, allowing the system to remain legible on any background. Every time a number is drawn, the colours are reshuffled — so the same number can look different while always following the same rules.
